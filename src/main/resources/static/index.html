MicroSpringBoot Server

Servidor Web Java con capacidades IoC - Taller de Reflexión

Endpoints Disponibles:

GET /
Probar endpoint raíz
GET /hello
Probar saludo
GET /pi
Ver valor de PI
GET /greeting?name=TuNombre
Saludo por defecto | Saludo personalizado

Características Implementadas:

  • Anotaciones personalizadas (@RestController, @GetMapping, @RequestParam)
  • Servidor HTTP que escucha en puerto 8080
  • Mapeo de rutas a métodos usando reflexión
  • Soporte para parámetros de query con valores por defecto
  • Servir archivos estáticos (HTML, PNG)
  • Escaneo automático de controladores con @RestController

Desarrollado con Java 21 - TDSE Project
